Tenziesweb game

CSS & React

GitHub Repo Live version

This game of chance, built using React, challenges users to roll the dice until they all match. Users can choose to freeze any die at its current value between rolls.

The application is optimized for mobile devices, using a Mobile-First design approach. The user interface is designed to be user-friendly and intuitive, making it easy to use on smartphones and tablets. The layout is responsive on all devices, with CSS Grid and Flexbox employed to create a dynamic layout that adapts to different screen sizes. This ensures a seamless user experience across all devices.
