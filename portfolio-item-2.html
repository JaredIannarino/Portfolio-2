Tenziesweb game

HTML, CSS, & JS

GitHub Repo Live version

This game of chance uses React and React Components to have the user roll until all dice are the same. The user can click each die to freeze it at its current value between rolls.

This app has a Mobile-First design, but is completely responsive on all devices. This site utilizes CSS Grid and Flexbox to layout to achieve a dynamic layout.
