This site uses Vanilla JavaScript and the Online Media Database API to allow th user to create media database of all of their favorite movies, tv shows, and video games.

This app has a Mobile-First design, but is completely responsive on all devices. This site utilizes CSS Grid and Flexbox to layout to achieve a dynamic layout.

Create the ultimate catalog of all your favorite visual media. Never get caught lacking when someone asks for your top 10 favorite movies again!
